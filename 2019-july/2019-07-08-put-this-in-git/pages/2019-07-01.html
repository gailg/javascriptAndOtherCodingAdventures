Block level elements and inline elements

Block level elements have height equal to its content and width equal to that of its container. So block elements always start on a new line. Block elements can be nested in other block elements, and they can wrap in-line elements. Paragraphs, divs, and headings are block elements.

Inline elements have height and width equal to its contents. Elements align left, side by side, in a line. Anchor tags, span, and strong are examples of block elements.

The box model

Pixels are fixed sizing. Percentages are used to size elements relative to their containers. Width and height are used to change the default size of elements. Padding is used to adjust the space inside of the element. Margin is used to adjust the space outside the element. The border displays between the padding and the margin.
